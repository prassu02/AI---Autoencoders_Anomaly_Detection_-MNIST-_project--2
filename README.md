
# 📦 Autoencoder-Based Anomaly Detection (MNIST)

## 🚀 Overview
This project implements an anomaly detection system using deep autoencoders on the MNIST dataset. The model learns normal patterns and identifies anomalies based on reconstruction error.

---

## 🧠 Features
- Deep learning-based anomaly detection
- Uses autoencoders for reconstruction
- Detects anomalies using reconstruction error threshold
- Works on MNIST image dataset
- Visualizes normal vs anomalous samples

---

## 🛠️ Tech Stack
- Python  
- TensorFlow / PyTorch  
- NumPy  
- Matplotlib  
- Scikit-learn  

---

## 📊 Performance
- Achieved ~95% anomaly detection accuracy  
- Reduced training time by ~28% with optimized pipeline  
- Efficient reconstruction error-based detection

## 🔍 How It Works
- Train autoencoder on normal MNIST digits  
- Reconstruct input images  
- Calculate reconstruction error  
- Detect anomalies when error exceeds threshold
